Git homework - User manual

git config --global user.name ""

This command sets your global username. Typically you should do it after instaling Git, but you can change it in any time. This is important, because every your commit must contain this information. The command sets your username for all your future commits, but if you want to set other username for a some project you can use this command without "--global" in the project directory.

gitk -all

gitk by itself displays changes in a repository or a selected set of commits(includes visualizing the commit graph, showing information related to each commit, and the files in the trees of each revision).
Option -all shows all refs (branches, tags, etc.).
Sources: https://git-scm.com/docs/gitk

git add .

git add adds modified files to the Git index (staging area, like a queue). Those files will be commited later.
Option '.' in Git 2.X is the same as [-A/--all] and adds all files in entire working tree.
Source: https://git-scm.com/docs/git-add
